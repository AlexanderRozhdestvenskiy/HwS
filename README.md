# HwS

Курс Пола Хадсона [100 Days of SwiftUI](https://www.hackingwithswift.com/100/swiftui).
Первые 14 дней посвящены основам языка Swift.
После этого начинаем создавать учебные приложения с помощью фреймворка SwiftUI. Все приложения представлены ниже.

## 1. WeSplit

В этом проекте мы создали приложение для разделения чеков, которое можно использовать после еды в ресторане - вы вводите стоимость своей еды, выбираете, сколько чаевых вы хотите оставить и сколько людей вы с вами, и оно сообщит вам, сколько нужно заплатить каждому человеку. Что использовано: Form, NavigationView, @State и другое.
